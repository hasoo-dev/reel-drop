## ✨ Key Features

* **Direct Downloads:** Paste a URL and automatically detect the highest available quality stream to immediately trigger the download. No tedious manual quality selection required.
* **High-Quality Support:** Optimized to fetch and download videos in the best possible resolution, including 1080p for platforms like Pinterest.
* **Responsive Design:** Seamless experience across both mobile and tablet devices with an optimized, proportional scaling architecture.
* **Built-in Scanner:** Integrated camera functionality to quickly scan links or codes.
* **Clean & Modern UI:** A professional and intuitive user interface built with the user in mind.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have the Flutter SDK installed. If not, follow the official [installation guide](https://docs.flutter.dev/get-started/install).

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/reel-drop.git
   ```
2. Navigate to the project directory
   ```bash
   cd reel-drop
   ```
3. Install dependencies
   ```bash
   flutter pub get
   ```
4. Run the app
   ```bash
   flutter run
   ```

## 📂 Project Structure

A quick overview of the key directories in the project:

* `lib/app/`: Contains the core application views and logic (e.g., `home_view`, scanner, etc.).
* `lib/services/`: Services for handling API calls, downloads, and other core functionalities.
* `assets/`: Contains static assets like images, icons, and fonts.
